# Advent 2023
Solutions for the [2023 Advent of Code](https://adventofcode.com/2023) challenge

### Setup
This assumes you have [Pipenv](https://pipenv.pypa.io/en/latest/) installed.
```
cp .env.example .env
make install
```

### Running code
```
DAY={day integer} make run
```
i.e.
```
DAY=2 make run
```

### Running lint and tests
```
make lint
make test
```
